🟢 Energy & Health – ASP.NET MVC Web Application
📌 Description

Energy & Health is a three-layer ASP.NET MVC web application developed as a diploma project for acquiring a third degree of professional qualification in the specialty System Programming.

The system represents an online platform for distribution of food supplements. It allows customers to browse products by category, place orders online, and manage their shopping cart. Administrators can manage products, categories, brands, and orders through an admin panel.
